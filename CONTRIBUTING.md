# Contributing

## Branching
- main: stable
- develop: ongoing work
- feat/*: features
- fix/*: bug fixes
- chore/*: maintenance

## Workflow
1) Create a branch from develop
2) Commit changes with clear messages
3) Open a PR into develop
4) Merge after review / CI passes

## Commit style (suggested)
- feat: ...
- fix: ...
- chore: ...
